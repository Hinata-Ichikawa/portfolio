tabitt

ハッカソン：2017.10.7 - 2017.10.8　個人制作：2017.10 - 2017.12

使用したスキル：HTML / CSS / JavaScript Ruby on Rails

OthloHack2017に参加しました。ハッカソン自体初参加のスキルもバラバラな5人が集まり、かなり苦戦しました。 tabittというWebアプリを制作しましたが、結果、2Daysでは完成せず... 悔しかったのでRuby on Railsを1から勉強し、tabittを完成させました。 というお話です。

人生初のハッカソンに参加

大学2年生の秋、個人で勉強することの多かった私はアウトプットの機会を探していました。

そんな時にOthloTech運営の先輩から、このOthloHack2017を紹介してもらい、せっかくということで参加しました。

旅を共有できるアプリに決定

大学の空きコマ、会社でのちょっとした休憩時間に、地元の人しか知らないちょびっとした旅を投稿し共有できるWebアプリ『tabitt』を制作することに。

使用言語はHTML,CSS,Rubu on Rails。私はHTML,CSSを担当しました。コンセプトが決まったのは1日目の深夜、カラオケでのことでした...。

ちなみに『tabitt』はちょびっとした旅から命名されました。

しかし、2Daysで完成せず。

徹夜で制作していたのはよかったものの、翌朝始発の時間に一時解散。2日目の朝、集合時間にいたのは私だけという悲惨な事態に(笑)。このチームはみんな死ぬほど朝に弱かったです...。

途中でメンバーは揃いましたが、私はRailsの知識が皆無だったため、組み込みをすることができず、そのままタイムオーバー。

頑張っても完成しないなら意味がない。うん、よし、『tabitt』作ってやるってこの日に決意しました。

とりあえずRails勉強。そして...

tabitt制作には必須のRuby on Railsをまずは勉強せねば、ということでProgateで徹底的に基礎を叩き込む。

Progateは私のスキルの基礎を固めてくれた良き存在です...。とりあえずRailsのレッスンを2周して、スライドを見ながら制作。初めての言語にエラー連発。

2ヶ月かかりました。でもちゃんと完成しました。
